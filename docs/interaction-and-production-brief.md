# 互动设计

## 开场互动
- 弹幕：你有没有遇到 AI 重复犯同一个错误？
- 画面：三个按钮式选项：忘规则 / 忘 bug / 忘项目背景

## 中段互动
- 弹幕：你觉得看板里最多的噪音是什么？
- 选项：系统注入 / 进度播报 / 探测命令 / 临时文件

## 结尾互动
- 评论区投票：下一期做哪个？
  1. 自动导出 Markdown 日报
  2. 自动生成周报
  3. 接入更多 Agent 工具
  4. 做一键安装器

## CEO 视频制作要求
- 需要配音。
- 需要中文字幕。
- 需要少量互动弹幕设计。
- 需要高级展示感：暗色主题、数字放大、架构图、功能演示切换。
- 最后输出完整视频，而不是单独片段。
