## 简介
这期分享一套我自己搭的本地 Codex 工作流：

- AGENTS.md 固化项目规则和编码习惯
- agent-memory-mcp 保存 bug 修复、技术决策和长期记忆
- LEARNINGS.md 记录人类可读的踩坑台账
- 每日产出看板自动读取本地 Codex sessions
- 支持按项目筛选、核心内容过滤、跨天会话归档
- Windows 可双击桌面图标打开，零依赖 Node 服务

重点不是做一个花哨面板，而是让 AI 编程助手把“踩过的坑”真正记住。
