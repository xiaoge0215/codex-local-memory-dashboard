## 简介正文

给 Codex 装一个本地「记忆脑 + 自我进化 + 每日产出看板」，不依赖云端，数据全部留在本地。

- 本地记忆：AGENTS.md 固化规则 + agent-memory-mcp 保存长期记忆
- 自我进化：LEARNINGS.md 把 bug 修复变成可复用经验
- 每日看板：按天/按项目/核心内容过滤，自动整理 Codex 每天产出
- 低配可用：零依赖 Node 服务，按需启动

GitHub 项目（含代码、模板、视频脚本）：
https://github.com/xiaoge0215/codex-local-memory-dashboard

一键启动：双击 tools/daily-dashboard/看板.bat，打开 http://localhost:3455
